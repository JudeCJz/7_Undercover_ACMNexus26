🛡️ Link Safety Scanner: Test Harness

Use this page to verify that the Sentinel One extension correctly identifies various security threats locally on your system.

🚩 Blacklisted Domains

These domains are pre-written into the default database in background.js. They should always be highlighted in red.

  • Direct match for malicious domain
    Dangerous Link (malicious.com)
  • Direct match for phishing domain
    Phishing Login (phishing-site.net)
  • Direct match for bad-actor.org
    Exploit Link (bad-actor.org)

🔍 Suspicious Keyword Heuristics

These links go to external domains and contain trigger keywords like "verify", "secure", or "login".

  • External link with "verify" and "account"
    Verify Your Account Now
  • External link with "secure" and "login"
    Secure Login Portal
  • External link with "update"
    Urgent Software Update

✅ Safe Links

These domains are in the whitelist (google.com, github.com) or are standard harmless sites. They should NOT be highlighted.

  • Whitelisted domain
    Search on Google
  • Whitelisted domain
    Check GitHub Repo
  • Internal or fragment link
    Back to top
